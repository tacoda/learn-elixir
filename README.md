# Learn Elixir

## Basics

- `iex`
- `elixir`
- `elixirc`

To exit `iex`, `Ctrl+C` twice

To run `iex` with `SCRIPTNAME` loaded, `iex -S SCRIPTNAME`

IO.puts "Hello world from Elixir"
```

$ elixir simple.exs

```
